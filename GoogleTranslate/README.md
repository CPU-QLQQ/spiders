# Translate
本脚本用于将 [U.S. DEPARTMENT OF AGRICULTURE FoodData Central Data](https://fdc.nal.usda.gov/download-datasets.html) 提供的 csv 数据并进行翻译，翻译部分调用 Google Translator API 实现。
